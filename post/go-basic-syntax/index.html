Go Programming Language Basic Syntax

In this series, I will list some notes about Go Programming Language syntax.

数组

初始化

  1. 指定所有元素

  2. 自动推断数组长度

    b1 := [...]int{4, 5, 7}

  3. 指定下标

    var a1 [100]int
a1 = [100]int{2: 3, 5}
    1. 和 1) 可以混合使用

多维数组的初始化

var aa1 [3][2]int
aa1 = [3][2]int{
        [2]int{1, 2},
        [2]int{4, 5},
}
  • 数组是值类型，赋值会拷贝
  • 数组支持 `==` 和 `!=` 操作
